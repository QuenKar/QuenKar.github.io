IDEA推荐插件及相关配置

2020-09-27 1 min read # IDEA

Material Theme UI

Atom Material Icons

皮肤插件，还是十分好看的，直接上图

不过有一点不好，注释的颜色是浅灰色的，不是很明显，我换成了深绿色（#069252）

CodeGlance

类似与vscode的侧边栏，代码缩略滚动，还可以直接显示鼠标停留的那段代码

Codota

可以直接在IDEA中查询某个类以及相关方法的使用，给出的例子也都不错。

Ctrl+Shift+y显示查询框

以String类举例：

IDE Features Trainer

推荐学习IDEA使用教程，主要是快捷键方面的使用。假如之前没用过还是值得学一下，可以提高编程效率哦。

（英文教程，需要一些英语知识哦）

Key Promoter X

IDEA快捷键记录，会提示相关功能的快捷键提示，还会显示个人常用的快捷键，有时候忘记了一些快捷键的使用，可以直接去翻翻。

Rainbow Brackets

嵌套括号彩色显示，可以更好的区分代码块

以上就是我使用的一些插件，over

      • Material Theme UI
      • Atom Material Icons
      • CodeGlance
      • Codota
      • IDE Features Trainer
      • Key Promoter X
      • Rainbow Brackets
