ZSH+oh-my-zsh配置

2020-09-24 1 min read # 环境配置

环境：deepin（Ubuntu也通用）
1.安装zsh
sudo apt-get install zsh
2.切换bash为zsh
chsh -s /bin/zsh
如果要切回bash就：
chsh -s /bin/bash
3.安装oh-my-zsh
从GitHub上面拉取下来，放在~/.oh-my-zsh目录下
git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
备份已有的zshrc, 替换zshrc
cp ~/.zshrc ~/.zshrc.orig
cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
配置主题：
vim ~/.zshrc
推荐ys这个主题：
ZSH_THEME="ys"
