Vue常用命令

2021-01-04 1 min read # Vue

淘宝镜像

--registry=https://registry.npm.taobao.org

vue-cli

下载vue-cli

npm install -g vue-cli

删除vue-cli

npm uninstall vue-cli -g

@vue/cli

npm install -g @vue/cli

安装相关组件

vue-router

npm install vue-router

vuex

npm install vuex --save

axios

npm install axios --save-dev

Element-ui

npm i element-ui -S
